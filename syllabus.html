Course Description

This course introduces the operation of computer networks and networking devices. The course also provides an examination of the evolution of data communications and computer networks. Hardware and software network configurations, operations, and requirements will be discussed. Topics include network media, physical and logical topologies, the OSI model, protocols and standards, network implementation, and network support. This course is designed to prepare the successful student for the Network+ certification. Fee is required. (4 contact hours)

Course Level Outcomes

  • Describe history and evolution of data communications and computer networks
  • Determine hardware and software requirements to operate various types of networks
  • Describe the various types of data transmission media topologies
  • Describe Ethernet media access
  • Categorize network operations in terms of the OSI model
  • Discuss various types of protocol stacks and their advantages and disadvantages
  • Analyze various types of network configurations
  • Use various TCP/IP utilities to troubleshoot and analyze IP networks
  • Describe the purpose and characteristics of fault tolerance
